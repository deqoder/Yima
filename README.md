# Yima(意码）
# 总体目标
1. 分5到6类
2. 每键1个主字根，8个以内子字根，子字根数量为偶数，保持对称
3. 按含义及频率归并
# 大类-主字根
1. 五行：金木水火土
2. 生物：艸竹虫鱼犬
3. 人体：手足子女人
4. 五官：眉目耳鼻口
5. 工具：衣食方车文
